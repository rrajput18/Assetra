Supabase Integration Required

To run Assetra, you need to connect your remote Supabase database. Please follow these simple instructions:

1. Execute SQL Schema: Open the **SQL Editor** in your Supabase dashboard, create a blank query, copy/paste the SQL code from implementation_plan.md, and click **Run**.
2. Add Credentials: Open the .env file in your project folder and add your Project URL and Anon API key: 
VITE_SUPABASE_URL=your_project_url
VITE_SUPABASE_ANON_KEY=your_anon_key
3. Restart Dev Server: Stop the running dev server in your terminal (press Ctrl+C) and run npm run dev again to apply configuration changes.

Once you restart the server, the app will automatically connect and seed the default demo property!

Technologies Used

Vite & JS ES6+ Supabase DB Razorpay API Vanilla CSS3

Assetra is powered by a high-performance serverless architecture. **Vite** builds and compiles responsive ES6 modules, while a remote **Supabase Postgres** instance provides real-time transactional database integrity. Online payments are secured using the **Razorpay Standard Checkout SDK** with direct multi-tenant deposit routing.
